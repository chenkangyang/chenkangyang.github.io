Tutorial 建站教程

@Kangyang Chen · May 4, 2019 · 2 min read

建站教程

安装

参考 官方教程

Step 1 安装Hugo

首先安装 Hugo, 一个基于Go语言编写的一套静态博客生成器, 以macOS为例

$ brew install hugo
$ hugo version # 检查是否安装成功
$ cd ~/Sites
$ hugo new blog
$ cd blog

Step 2 安装主题

See themes.gohugo.io for a list of themes to consider. 这里使用了的主题为 hugo-nuo

$ cd themes
$ git clone https://github.com/laozhu/hugo-nuo

在 themems/hugo-nuo/exampleSite是一个网站的例子，在blog目录下我们也按照exampleSite中的结构来精简网站目录 若想自己自定义主题，则在blog目录下创建和themes/hugo-nuo下同名同位置的文件以覆盖主题中的内容。具体见该主题 README.md

$ brew install tree

在blog下使用 tree 命令生成网站结构

│   └── default.md
├── config.toml # Hugo编译选项以及网站的配置

Step 3 发布文章

$ hugo new posts/my-first-post.md

文章的顶部用 两行---包围的是yarn的配置信息，例如

---
title: "Construction Record"
author: "Kangyang Chen"
cover: "/media/cover.jpg"
tags: ["Hugo"]
date: 2019-05-03T23:41:42+08:00
draft: false
---

<!--more-->以上是文章内容的预览，其和cover会一同出现在首页中

文件中若要引用 media中的媒体，路径如同：/media/cover.jpg

本地预览

Start the Hugo server with drafts enabled:

$ hugo server -D

以上命令会在localhost:1313启动本地服务器来预览内容，文章的yarn配置中若draft:ture，则执行

$ hugo server

不会预览该草稿文章

部署

由于在config.toml中已经配置过 theme, baseURL等信息， 因此生成新的静态站点时候，可以不添加如这样的后缀: $ hugo --theme=hyde --baseUrl="http://chenkangyang.github.io/"

直接：

$ hugo

生成的静态站点在public中, 可以将其上传到GithubPages

$ cd public
$ git init
$ git remote add origin https://github.com/chenkangyang/chenkangyang.github.io
$ git add -A
$ git commit -m "first commit"
$ git push -u origin master

后续可以将媒体文件放到到云端，减轻GithubPages的压力

如：VSCode中安装插件Picgo配置markdown中的图片自动上传到七牛云中并回传markdown格式的图片外链
