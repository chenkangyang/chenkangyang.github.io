使用c语言实现一个词法分析器

@Author Name · May 4, 2019 · 2 min read

c语言实现一个c语言的词法分析器，对单词进行识别和编码

要求

                   +-------+                      +--------+
-- source code --> | lexer | --> token stream --> | parser | --> assembly
                   +-------+                      +--------+

设计并实现一个词法分析器，实现对指定位置的类C语言源程序文本文件的读取，并能够对该源程序中的所有单词进行分类，指出其所属类型，实现简单的词法分析操作。 - 允许用户自己输入源程序并保存为文件 - 系统能够输出经过预处理后的源程序（去掉注释、换行、空格等） - 能够将该源程序中所有的单词根据其所属类型（整数、保留字、运算符、标识符等。定义的类C语言中的标识符只能以字母或下划线开头）进行归类显示，例如：识别保留字：if、int、for、while、do、return、break、continue等，其他的都识别为标识符；常数为无符号整形数；运算符包括：+、-、*、/、=、>、<、>=、<=、!=等；分隔符包括：,、;、{、}、(、)等。 - 实现文件的读取操作，而不是将文本以字符串形式预存于程序中。文本内容为待分析的类C语言程序。

例如下面为一段C语言源程序：

main()
{
int  a,b;
a = 10;
   b = a + 20;
}

要求输出如下:(可以自行分类)

（5，’（’）
（5，’）’）
（5，’{ ’）
（1，’int’）
（2，’a’）
（5，’,’）
（2，’b’）
（5，’;’）
（2，’a’）
（4，’=’）
（3，’10’）
（5，’;’）
（2，’b’）
（4，’=’）
（2，’a’）
（4，’+’）
（3，’20’）
（5，’;’）
（5，’}’）

设计

  1. 完成文件读取
  2. 文本识别 识别为5类符号：标识符、保留字、常数、运算符、界符 定义5类符号的集合为： 1：保留字(32) auto break case char const continue
    default do double else enum extern
    float for goto if int long
    register return short signed sizeof static
    struct switch typedef union unsigned void
    volatile while 2：标识符 letter(letter | digit)* 无穷集 3：常数(无符号整形数) (digit)+ 无穷集 4：运算符 <、<=、>、>=、=、+、-、*、/、^、等 5：界符 ‘/*’、‘//’、 () { } [ ] “ ” ‘ 等
  3. 状态转换图
